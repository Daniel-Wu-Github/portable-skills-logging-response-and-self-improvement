## Pattern Entry Format (for manual additions)

```

- **Last seen:** YYYY-MM-DD
- **Occurrences:** N
- **Description:** full description of the pattern
- **Root cause:** what was actually wrong

- **Skill gap:** why the skill didn't prevent it
- **Resolution applied:** what fix was used
```
